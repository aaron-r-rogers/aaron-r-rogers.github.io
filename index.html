Skills

As a software engineer, it's difficult to summarize all of the tools in my kit, and I'm always learning new things. This is a list of some of the skills and technologies that I identify as strengths and enjoy including in my stack.

JavaScript

Oh the language of the internet. It's foundational to most of my work, and I appreciate its flexibility -- there are many ways from point A to point B.

React.js

React is not without its flaws, but it sure makes for a gorgeous user experience and supports some developer-friendly principles.

Redux

React is all about state management, and storing it all in one place where any component can access it improves performance, debugging, and ease of development.

Node.js

If you're already using JavaScript on the front end, why not use it on the back end? Node supports API services with asynchronous event-based exchanges of data.

PostgreSQL

PostgreSQL is a powerful system for managing relational databases. The key is that it features transactions with Atomicity, Consistency, Isolation, Durability (ACID) properties

Microsoft .NET

Besides Node, another way I can set up a server is using the C# language and Microsoft's .NET framework. Combining object-orjented programming with MVC architecture, you know it's going to perform and play well with any platform.
